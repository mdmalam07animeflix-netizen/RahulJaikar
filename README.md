# Anime Downloader Bot

Professional anime delivery bot.

## Features
- Channel post → Download
- ₹99 subscription
- Donation (any amount)
- 3-line admin panel
- 24/7 with Render + UptimeRobot
- Zero storage

## Deploy
1. Connect GitHub to Render
2. Add Environment Variables
3. Use UptimeRobot for 24/7
